# Basic-Node-Express-Server
Basic node express server for getting started.

## Isolated Clone
```
npx degit https://github.com/suhailtajshaik/basic-node-express-server.git PROJECT_NAME
```
## Running 
> One time installation:
```
npm install
```
> ### development mode :
```
npm start
```
> Production mode 
```
node index.js
```
